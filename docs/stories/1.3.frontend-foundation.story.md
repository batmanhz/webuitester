# Story 1.3: Frontend Foundation & Layout
**Status**: Draft

## Story
**As a** Frontend Developer,
**I want** to set up the Vue 3 application structure, install UI libraries (Element Plus), and implement the core Main Split View layout,
**so that** we have the visual shell for the "IDE-like" interface.

## Acceptance Criteria
1.  Vue 3 + Vite project structure organized (`src/components`, `src/views`, `src/stores`).
2.  **Element Plus** installed and configured (Dark mode enabled by default).
3.  **Tailwind CSS** (or just custom CSS) installed for utility classes (optional, but recommended for layout).
4.  **Main Layout** implemented matching `front-end-spec.md` (Sidebar + Content Area).
5.  **Split View Component** integrated (using `splitpanes` or similar) for the Test Editor page.
6.  **Navigation** (Vue Router) configured with routes:
    *   `/` (Home/Test Cases List)
    *   `/case/:id` (Editor/Split View)
    *   `/settings` (Settings)

## Tasks / Subtasks
- [ ] Install Frontend Dependencies
    - [ ] `npm install element-plus @element-plus/icons-vue`
    - [ ] `npm install splitpanes` (for the resizable split view)
    - [ ] `npm install sass` (Element Plus often requires sass)
    - [ ] `npm install vue-router@4 pinia` (if not already added)

- [ ] Global Styles & Theme [Source: front-end-spec.md#3]
    - [ ] Configure Dark Mode as default.
    - [ ] Set Primary Color to `#8B5CF6` (Vibrant Purple) in Element Plus theme config.
    - [ ] Set global typography (Inter/Roboto).

- [ ] Project Structure Setup
    - [ ] `src/layouts/MainLayout.vue` (Sidebar + Router View)
    - [ ] `src/views/HomeView.vue` (Placeholder for Case List)
    - [ ] `src/views/EditorView.vue` (The main IDE view)
    - [ ] `src/views/SettingsView.vue` (Placeholder)
    - [ ] `src/router/index.ts` (Route definitions)

- [ ] Implement Main Split View (EditorView.vue) [Source: front-end-spec.md#5.1]
    - [ ] Integrate `splitpanes` to create a 50/50 resizable layout.
    - [ ] **Left Panel**: Create placeholder `SmartEditor.vue`.
    - [ ] **Right Panel**: Create placeholder `ExecutionFeedback.vue`.
    - [ ] Add basic styling to visually distinguish panels (e.g., borders, background colors).

- [ ] Basic Navigation (Sidebar)
    - [ ] Implement `el-menu` with links to Home, Settings.

## Dev Notes
### UI/UX Alignment
*   **Theme**: The spec calls for "Dark Mode (Default/Only)". Ensure `html` tag has `class="dark"` or Element Plus dark theme provider is used globally.
*   **Split View**: This is the core of the UX. Use a robust library like `splitpanes` to ensure smooth resizing.
*   **Color Palette**:
    *   Primary: `#8B5CF6`
    *   Background: Dark gray/black (standard Element Plus dark theme).
